🔬 微生物醫檢師冒險：臨床鑑定模擬器 (Microbe MT Adventure)
這是一個專為醫檢學生與實習醫檢師設計的互動式網頁遊戲，旨在模擬真實臨床微生物實驗室的鑑定決策流程。使用者必須扮演醫檢師，在有限的資源與壓力下，從檢體接收、培養基選擇到最終報告提交，完成完整的鑑定路徑。
🌟 核心特色
1. 臨床路輯仿真
多階段決策： 嚴格遵循臨床標準作業程序，包含「培養基接種」、「染色判讀」與「生化鑑定」三大階段。
動態鑑定路徑： 系統會根據使用者的選擇提供對應的影像資訊，例如 Gram Stain 鏡檢影像或生化反應管結果。
2. 技術開發亮點
純前端架構： 無需後端伺服器即可執行，利用 JavaScript 處理所有遊戲邏輯與流程狀態。
安全評分機制： 實作自定義的加密模組（結合 XOR 運算與 Base64 編碼），保護本地端排行榜資料不被輕易竄改。
現代化 UI/UX： 採用響應式深色模式設計，並加入動態進度條與通知系統（Notification Overlay），提升操作回饋感。
3. 容錯與風險評估
專業指標系統： 引入「專業分數」概念，錯誤的決策（如選錯培養基或鑑定錯誤）會導致失分，甚至吊銷執照（End Game），模擬臨床工作的嚴謹性。
🛠️ 技術細節
開發語言： Vanilla JavaScript (ES6+), HTML5, CSS3。
資料結構： 使用對象映射（Species Map）管理複雜的微生物分類與鑑定關聯。
持久化儲存： 利用 localStorage 搭配加密演算法實現玩家紀錄保存。
📖 鑑定流程演示
檢體分析： 閱讀病歷描述與檢體類別（如：Urine, Pus）。
培養基接種： 從 BAP, EMB, CHO 等常用培養基中選出正確組合。
初步鑑定： 觀察菌落形態並進行革蘭氏染色（Gram Stain）判讀。
進階鑑定： 自主選擇質譜儀（Mass Spec）、自動化儀器（VITEK 2）或傳統生化試驗。
提交報告： 確認最終菌種名稱並獲取治癒成就。
